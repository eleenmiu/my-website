# glfekq.com UI 风格分析与转译

参考站：https://www.glfekq.com/

## 风格关键词

- 医疗连锁品牌感：黑色顶部品牌条、白色大导航、中心 Logo，强调“品牌年限、医生团队、连锁机构、服务数据”。
- 蓝绿医疗主色：高频使用 `#02A4B8` 青蓝，搭配绿色渐变，传达洁净、科技、可信赖。
- 信息密度偏高：首屏之后快速进入品牌介绍、五大诊疗中心、医生团队、案例数据和资讯入口。
- 中英双语层级：中文作为主信息，英文小标题如 `Doctor Team`、`5 Distinctive D&T Centers` 作为栏目气质。
- 强转化入口：参考站有弹窗、右侧商务通、快捷咨询；本原型转译为顶部电话、预约条和右侧浮动快捷入口。

## 版式规律

- 顶部结构：黑色细顶栏 + 白色大导航，导航项为中文主标题和英文副标题。
- 首屏结构：宽幅医疗图片或活动 Banner，视觉占比大，底部露出后续内容。
- 内容节奏：每个区块大约 60-80px 上下留白，中心容器宽度约 1180-1200px。
- 卡片系统：诊疗中心使用图片卡片和渐变遮罩，医生团队使用大图卡片叠加姓名与科室标签。
- 数据背书：案例数据使用浅红、浅黄、浅蓝卡片，与医疗青绿主色形成柔和对比。

## 本次 UI 处理

- 保留参考站的“医疗集团首页”结构，但弱化强促销弹窗，让页面更现代、清爽。
- 将快捷咨询从弹窗改成页面内预约工具条，同时保留右侧浮动入口，提高转化但不遮挡内容。
- 使用原创品牌名“深惠口腔”和外部医疗影像素材，避免直接复刻目标站图片。
- 保留青蓝 + 绿色渐变、黑白顶栏、英文栏目标题、五大诊疗中心、医生团队、案例数据这些关键识别点。

# 深惠口腔客资管理 App

## 本地启动

```bash
node crm-server.js
```

官网入口：

```text
http://127.0.0.1:8090/
```

后台入口：

```text
http://127.0.0.1:8090/admin.html
```

默认账号：

```text
admin / 123456
```

可以用环境变量修改账号密码：

```bash
CRM_USER=shenhui CRM_PASS='你的强密码' node crm-server.js
```

## 已支持

- 后台登录
- 查看客户姓名、电话、咨询项目、来源、提交时间
- 搜索和状态筛选
- 客资状态：新客资、已联系、已预约、无效
- 跟进备注
- 手动录入客资
- 官网表单和正畸预约表单自动进入后台
- 后台在线时，新客资会弹出页面提醒
- 点击“开启通知”后，可使用浏览器系统通知

## 数据位置

本地客资保存到：

```text
data/leads.json
```

## 上线说明

GitHub Pages 只能托管静态网页，不能运行登录、数据库和通知服务。
这个客资 App 需要部署到一台支持 Node.js 的服务器，或后续改成云数据库/Serverless。
